
# PayGate CLI example

WIP

A node.js example CLI to interact with the HTTP REST service of [moov-io's Paygate](https://github.com/moov-io/paygate).

[![gif with examples][examples-link]][examples-link]

# Running

```
$ npm install
$ node bin/main.js
```
